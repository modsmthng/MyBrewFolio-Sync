# Security Policy

## Supported versions

MyBrewFolio Sync is experimental. Security fixes are provided for the latest published release.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability. Send a concise report to
`security@mybrewfolio.com` with the affected version, platform, reproduction steps, and expected
impact. Do not include OAuth tokens, private library data, local network addresses, or other users'
data.

Reports are acknowledged as soon as practical. Please allow time for validation and a coordinated
fix before public disclosure.
