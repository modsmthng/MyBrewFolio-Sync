# Contributing

Thank you for helping improve MyBrewFolio Sync.

## Before opening a pull request

1. Open an issue for substantial behavioral or protocol changes.
2. Keep the companion one-way from GaggiMate to MyBrewFolio.
3. Never add telemetry, secrets, real user library fixtures, or public-network machine targets.
4. Add or update tests for parser, protocol, storage, or synchronization changes.
5. Run the complete verification suite documented in `README.md`.
